星屑救助隊 共有版

公開URL:
https://ken0592.github.io/roguelike/

目的:
ランダム生成される「星喰いの塔」をB1FからB100Fまで踏破します。
9・19・29...99階に門番ボス、10階ごとに休憩所、20階ごとに大休憩所があります。
ボス撃破時はレリックを3つから1つ選べます。

成長:
技を使うと経験値を得られます。
物理・魔法の使用回数、レベル、到達階、変異種撃破、素材で進化先が分岐します。
進化した姿、素材袋、バッグ拡張は次の挑戦へ引き継がれます。
レベル、レリック、星網スキル盤は挑戦ごとに初期化されます。

主な操作:
移動: WASD / 矢印キー / テンキー
通常攻撃: 左クリック / F
選択中の技: 右クリック / J
技メニュー: Q
月蜜の実: R
待機: Space / .

iPhone:
Safariで公開URLを開き、共有から「ホーム画面に追加」を選びます。
「Webアプリとして開く」をオンにすると横画面アプリとして起動できます。
左の十字キーで8方向移動、Aで通常攻撃、Bでダッシュ、Xでメニュー、Yで技を使います。

セーブ:
3つの枠に記録できます。
10階ごとの休憩所では、次の区画から再開できる記録が自動保存されます。

素材:
Kenney Roguelike/RPG Pack (CC0)
キャラクターアート: OpenAI生成
