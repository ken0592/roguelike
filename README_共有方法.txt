星屑救助隊 共有版

遊び方:
1. index.html をブラウザで開きます。
2. 3つのセーブ枠から、使用する記録を選びます。
3. 星見広場のダンジョン門で次の迷宮を選びます。
4. 商店・倉庫・道場で準備し、出発門からダンジョンへ向かいます。

攻略:
5つのダンジョンを、すべて固有ボス戦まで遊べます。
階層数は順に5階、7階、10階、15階、20階です。
各階の先遣員を救うか禁じられた決着を選び、最深部のボスを倒すと次の迷宮が解放されます。
制覇するたびに町へ戻り、買い物・倉庫・訓練を行えます。
開いた階段に乗ると、次へ進むか探索を続けるか選べます。

挑戦ごとの成長:
レベル、進化、蜘蛛型スキルボードは挑戦ごとに初期化されます。
武器・防具・お守りはQメニューの「装備」で付け替えられます。
拾った装備は町へ持ち帰り、次の挑戦にも引き継げます。
装備は星1コモンから星5レジェンドまでの5段階です。
RARE表示の敵を倒すと、星3以上の装備が確定します。

セーブ:
3つの枠に迷宮攻略、町の訓練、装備、カルマ、バッグ拡張を保存できます。
町の上部にある保存ボタンを使います。迷宮制覇時は自動保存されます。

主な操作:
移動: WASD / 矢印キー / テンキー（8方向）
向きだけ変更: Shift + 移動キー
通常攻撃: 盤面を左クリック / F
選択中の技: 盤面を右クリック / J
技メニュー: Q
月蜜の実: R
待機: Space / .

スマートフォン:
iPhoneのSafariで公開URLを開き、共有から「ホーム画面に追加」を選びます。
「Webアプリとして開く」をオンにすると、専用アイコンから全画面で起動できます。
横画面では左に十字キー、右にABXYが表示されます。
十字キー長押し: 連続移動
十字キーを斜めへ滑らせる: 斜め移動
A: 通常攻撃
Bを押しながら移動: ダッシュ
X: メニュー
Y: 選択中の技
十字キー中央: 待機
一度読み込むと、通信が一時的に切れてもゲームを起動できます。

成長:
Qメニューの「成長」で、蜘蛛の八本脚に配置されたスキルへポイントを使えます。

進化:
主人公は星の子コハク一体です。
レベル、敵撃破、救助、依頼人への行動、進化素材によって進化先が変わります。
救済・闘争・禁忌は二段階進化、智樹・星王は条件の厳しい強力な一段階進化です。

カルマ:
依頼人を救うと1減り、警告後も攻撃して倒すと3増えます。
カルマは進化条件に影響します。

バッグ:
初期20枠です。星の商店で最大40枠まで永久に拡張できます。

地形:
緑の回復苔: HPを2回復
赤い亀裂: 2ダメージ
