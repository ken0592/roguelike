ふたりの星屑ダンジョン 共有版

遊び方:
1. このフォルダをZIPから展開します。
2. index.html をダブルクリックしてブラウザで開きます。
3. 同じキーボードで二人プレイできます。

操作:
P1 ミナ: WASD / E / Space
P2 レン: 矢印キー / / / Enter

注意:
ZIPの中のまま index.html を開くと素材が読めないことがあります。
必ずフォルダを展開してから開いてください。

オンラインでURL共有したい場合:
このフォルダの中身を GitHub Pages、Netlify、Vercel などの静的サイトホスティングにアップロードしてください。
