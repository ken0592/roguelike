# 星屑救助隊

星の子コハク一体で、ランダム生成される「星喰いの塔」100階踏破へ挑むターン制ローグライクです。

## 塔のルール

- 9・19・29...99階に、次の休憩所を守る固有ボスが出現
- 10階ごとに休憩所、20階ごとに大休憩所
- 休憩所で全技PP回復、大休憩所ではHPと満腹度も全回復
- ボス撃破時に3つのレリックから1つを選択
- マップ、敵、道具、隠し罠、隠し階段、階の運勢は挑戦ごとに変化
- 歩くたびにHPが1回復
- 変異種は重装・魔脈・疾走・吸収・爆発の特性を持つ

装備システムはありません。レリックには物理、魔法、生存、探索などの効果があり、獲得したものはすべて同時に働きます。

## 成長

技は使うだけでも経験値を得られます。物理技と魔法技の使用回数、レベル、到達階、変異種撃破、進化素材によって進化先が変わります。

進化には一段階と二段階があり、進化した姿と能力は次の挑戦にも引き継がれます。レベル、レリック、星網スキル盤は挑戦ごとに初期化されます。

星網スキル盤は「物理」「生存」「魔法」の三系統です。低い段階から順につなぎ、今回の引きに合わせて戦い方を組み立てます。

## 操作

- `WASD`、矢印キー、テンキー: 8方向へ移動
- `Shift` + 移動キー: ターンを使わず向きだけ変える
- 盤面を左クリック または `F`: 通常攻撃
- 盤面を右クリック または `J`: 選択中の技
- `Q`: 技選択メニュー
- `1` から `4`: 技を直接選択
- `R`: 月蜜の実を食べる
- `Space` または `.`: 待機

## iPhone

Safariで公開URLを開き、共有メニューから「ホーム画面に追加」を選びます。「Webアプリとして開く」をオンにすると、横画面の専用アプリとして起動できます。

横画面では左に十字キー、右にABXYを表示します。Aは通常攻撃、Bを押しながら移動するとダッシュ、Xはメニュー、Yは選択中の技、十字キー中央は待機です。

HPバーとミニマップは画面上部に固定されます。一度読み込んだゲーム本体と画像は端末へ保存され、通信が一時的に切れても起動できます。

## セーブ

セーブデータは3枠です。最高到達階、永続進化、進化素材、バッグ拡張、町の訓練を保存します。10階ごとの休憩所では再開地点が自動保存されます。
