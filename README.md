# 星屑救助隊

星の子コハク一体で5つの迷宮へ何度も挑む、ターン制ローグライクです。すべてのダンジョンを固有ボス戦まで遊べます。

町「星見広場」で準備し、入るたびに地形、敵、道具、装備、運勢が変わるダンジョンを探索します。ボスを倒して町へ帰ると次の迷宮が解放されます。

迷宮は順に全5階、7階、10階、15階、20階です。深い迷宮ほど進化条件を満たしやすく、敵も強くなります。

## 遊び方

- `WASD`、矢印キー、テンキー: 8方向へ移動
- `Shift` + 移動キー: ターンを使わず向きだけ変える
- 盤面を左クリック または `F`: 向いている方向へ通常攻撃
- 盤面を右クリック または `J`: 選択中の技を使う
- `Q`: 技選択メニューを開く
- `1` から `4`: 技を直接選ぶ
- `E`: 次の技へ切り替える
- `R`: 月蜜の実を食べる
- `Space` または `.`: その場で待つ

## iPhoneアプリとして使う

Safariで公開URLを開き、共有メニューから「ホーム画面に追加」を選び、「Webアプリとして開く」をオンにします。ホーム画面の専用アイコンから、ブラウザの枠がない横画面アプリとして起動できます。

横画面では左に十字キー、右にABXYを表示します。十字キーは押し続けると連続移動し、指を斜めへ滑らせると8方向へ移動できます。Aは通常攻撃、Bを押しながら移動するとダッシュ、Xはメニュー、Yは選択中の技です。中央の点は待機です。

一度読み込んだゲーム本体と画像は端末へ保存されるため、通信が一時的に切れても起動できます。セーブデータは端末ごとに保存されます。

各階の先遣員を救うか、禁じられた決着を選ぶと階段が金色のゲートとして開きます。乗った時に「次の階へ進む」「まだ探索する」を選べます。最深部ではボスを倒すまで封印が解けません。

ミニマップは盤面と重ならない右側の専用欄に表示されます。緑の回復苔は踏むとHPが2回復し、赤い亀裂は2ダメージを受けます。

自分は水色、通常敵は赤、レア敵は金、道具は黄色の輪郭です。ミニマップには訪れた部屋と実際に歩いた通路だけが記録されます。

バッグは初期20枠です。月蜜の実、雫青の実、あかつきの種、静寂の珠など、この世界固有の道具を専用アイコンで見分けて使えます。商店で最大40枠まで永久に拡張できます。

各階には武器・防具・お守りが落ちています。レア度は星1コモン、星2アンコモン、星3レア、星4エピック、星5レジェンドの5段階です。Qメニューの「装備」から3部位を自由に付け替えられます。

挑戦開始時にレベル、進化、蜘蛛型スキルボードが初期化されます。装備は町へ持ち帰り、次の挑戦にも引き継げます。レベルアップで得たポイントを蜘蛛の八本脚のどこへ使うか、毎回違うビルドを組めます。

主人公はコハク一体です。未進化の期間を長く過ごし、レベル、戦い方、救助、カルマ、敵の進化素材によって五系統へ分岐します。

救済・闘争・禁忌の三系統には第二進化があります。智樹セージと星王アステラは、厳しい条件を満たして一度で強力な姿へ進化する系統です。

通常敵は20種族×5タイプの100種です。さらに迷宮ごとのレア敵が5種おり、金色の輪郭とRARE表示で見分けられます。倒すと星3以上の装備が確定します。

各階の依頼人を救うとカルマが1減ります。依頼人へ一度攻撃すると警告が出て、もう一度攻撃して倒すとカルマが3増え、禁忌進化の素材が残ります。

セーブデータは3枠です。迷宮の攻略状況、町の訓練、所持金、バッグ、バッグ拡張、装備、カルマを別々に保存できます。町では上部の保存ボタンから記録し、迷宮制覇時には自動保存されます。

## 町の施設

- ダンジョン門: 5つの迷宮から次の挑戦を選ぶ
- 星の商店: 道具の購入とバッグ拡張
- 風見倉庫: 道具を預ける
- 月影道場: 回復とコハクの鍛錬
- 救助隊本部: ランクと記録を確認する
