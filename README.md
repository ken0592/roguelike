# ふたりの星屑ダンジョン

ふたりで同じ画面を囲んで遊ぶ、協力ローグライクです。  
ミナとレンがそれぞれ別の役割を持ち、2つの紋章を起動して階段の封印を解きながら、地下深くを目指します。

## 遊び方

- 1P ミナ: `WASD` で移動、`E` で調べる、`Space` で待機
- 2P レン: `矢印キー` で移動、`/` で調べる、`Enter` で待機
- ミナは `灯紋`、レンは `剣紋` を起動できます
- 両方の紋章を起動すると階段が開きます
- 近くで戦うと `絆追撃` が発生し、離れすぎると絆が下がります
